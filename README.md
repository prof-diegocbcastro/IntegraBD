# Subindo o projeto

## Configuração

### Execute o comando o comando para baixar as dependências do projeto
```bash
    bash run.sh configure
 ```

### Build o projeto
```bash
    bash run.sh build
 ```

 ### Rodando o projeto
```bash
    bash run.sh up
 ```
